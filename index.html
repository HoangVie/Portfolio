将来志向

HTML, CSS, JavaScript, React JS を学び、実践する

HTML は、ハイパーテキスト マークアップ言語として、ウェブサイト上でハイパーテキスト構造を構築したり、画像、ビデオ、音楽などのデジタル ファイルを宣言したりする役割を果たします。 HTML を使った Web プログラミングを学ぶことで、Web プログラミングを極めたいと思うようになります。HTML マークアップ言語として記述された CSS スタイルおよびスタイル要素。複数の Web ページのフォーマットを同時に制御できるため、ユーザーは Web ライティングの時間と労力を節約できます。

HTML と CSS を学んだ後、JavaScript が最も人気のあるプログラミング言語であり、Javascript は給与が保証されており、仕事の機会が多いので、学ぶために Javascript を選択しました。ReactJS は、企業がより優れた UI 構造を備えたアプリを作成して、UX を強化および最適化するのに役立ちます。このテクノロジーを使用すると、ユーザー インタラクションを簡単に増やすことができ、クリック率とコンバージョン率も高くなります。
